# StayLy – Sistema de hotelarias

**StayLy** é um painel de administração para pousadas e hotéis, desenvolvido para gerenciar de forma integrada reservas, quartos, hóspedes, faturamento e comunicação.

---

##  Funcionalidades Principais

- **Dashboard Informativo**  
  Apresenta os principais indicadores do hotel:
  - Localização (endereço, cidade/estado)
  - IP do último acesso
  - Quartos disponíveis vs. total
  - Taxa de ocupação
  - Reservas do dia
  - Check-ins pendentes
  - Check-outs do dia
  - Avaliação média (em estrelas)

- **Seções de Gestão**  
  - Quartos
  - Reservas
  - Hóspedes
  - Relatórios
  - Estacionamento
  - Faturamento
  - Serviços
  - Galeria de Imagens
  - Configurações gerais
  - Check-in / Check-out

- **Comunicação com Clientes**  
  - Email Marketing
  - Chat com hóspedes
  - Notificações via SMS
  - Integração com website do hotel
  - Coleta de avaliações
  - Central de atendimento

##  Como Usar (Setup e Deploy)

1. Clone este repositório:
   ```bash
   git clone https://github.com/levyabreu/StayLy.git
````

2. Abra o arquivo `index.html` (ou equivalente) com seu navegador ou configure via servidor web (ex: Apache, NGINX).
3. Faça deploy no GitHub Pages para hospedar gratuitamente.
